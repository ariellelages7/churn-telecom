# Predição de churn em uma empresa de Telecom

Este projeto foi desenvolvido como parte do curso de Cientista de Dados e tem como objetivo prever o churn de clientes em uma empresa de telecomunicações utilizando técnicas de análise de dados e machine learning.

## Problema de Negócio
O churn de clientes representa um grande desafio para empresas de telecom. Antecipar quais clientes têm maior probabilidade de cancelamento permite a criação de estratégias de retenção mais eficazes.

## Etapas do Projeto
- Entendimento do problema e contexto de negócio
- Análise exploratória dos dados (EDA)
- Tratamento e pré-processamento
- Modelagem preditiva
- Avaliação dos resultados
- Conclusões e próximos passos

## Tecnologias Utilizadas
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Resultados
O modelo desenvolvido apresentou desempenho superior ao baseline e foi capaz de identificar padrões relevantes associados ao churn, demonstrando potencial para aplicação em cenários reais.
