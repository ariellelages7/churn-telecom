# Predição de Churn em uma Empresa de Telecom

Este projeto foi desenvolvido como parte do curso de Cientista de Dados com o objetivo de aplicar técnicas de análise de dados e machine learning para prever o churn de clientes em uma empresa do setor de telecomunicações.

A proposta do projeto é demonstrar, de forma prática, como dados podem ser utilizados para apoiar decisões estratégicas de negócio, especialmente em um contexto onde a retenção de clientes é fundamental.

---

## Problema de Negócio

O churn de clientes representa um dos principais desafios para empresas de telecom. A alta competitividade do setor faz com que clientes tenham facilidade para trocar de operadora, o que impacta diretamente a receita e aumenta os custos de aquisição de novos clientes.

Antecipar quais clientes possuem maior probabilidade de cancelamento permite que a empresa desenvolva estratégias de retenção mais eficientes, direcionando esforços para os perfis de maior risco.

---

## Objetivo do Projeto

O objetivo principal deste projeto é construir um modelo de machine learning capaz de prever se um cliente irá ou não entrar em churn, com base em informações contratuais, demográficas e financeiras.

Como objetivos secundários, o projeto busca:
- Explorar e entender o comportamento dos clientes;
- Identificar variáveis associadas ao churn;
- Avaliar o desempenho de um modelo preditivo em um cenário realista.

---

## Etapas do Projeto

O desenvolvimento do projeto seguiu as seguintes etapas:

- **Entendimento do problema e contexto de negócio**  
  Análise do cenário de churn e definição do objetivo do modelo.

- **Análise exploratória dos dados (EDA)**  
  Exploração dos dados para identificação de padrões, relações e possíveis problemas de qualidade.

- **Tratamento e pré-processamento**  
  Limpeza dos dados, tratamento de valores ausentes, codificação de variáveis categóricas e preparação para modelagem.

- **Modelagem preditiva**  
  Construção de um modelo baseline e de um modelo de Regressão Logística para predição de churn.

- **Avaliação dos resultados**  
  Avaliação do desempenho do modelo por meio de métricas adequadas para classificação e análise de visualizações.

- **Conclusões e próximos passos**  
  Interpretação dos resultados e discussão de possíveis melhorias futuras.

---

## Tecnologias Utilizadas

- **Python**: linguagem principal do projeto  
- **Pandas**: manipulação e análise de dados  
- **Scikit-learn**: construção e avaliação dos modelos de machine learning  
- **Matplotlib**: visualização de dados e resultados  

---

## Resultados

O modelo desenvolvido apresentou desempenho superior ao baseline, demonstrando capacidade de identificar padrões relevantes associados ao churn.  

Os resultados indicam que variáveis relacionadas ao tipo de contrato, tempo como cliente e valor do pagamento mensal exercem influência significativa na probabilidade de cancelamento, reforçando os insights obtidos durante a análise exploratória.
